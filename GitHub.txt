Polecenia:
1. Zaloguj się do GitHub
- Dołącz współpracownika
- Odtąd komunikujecie się tak, jakbyście się nie widzieli (czyli przez GitHub)
- Utwórz 2 repozytoria publiczne: Projekt_DP, Projekt_STD, obydwa z dołączeniem pliku readme
- W plikach readme wpisz wasze imiona i nazwiska

2. W repozytorium Projekt_DP nanieś odpakowaną zawartość dp0.zip
- Stwórz Pull request domagające się dodania bibliotek
- W obu plikach state.cpp i builder.cpp dodaj pierwszą linę: #include "stdafx.h"
- Obydwa zaktualizowane pliki nanieś do właściwej gałęzi GitHub
- Stwórz Pull request domagające się dodania informacji o wersji
- W obu plikach state.cpp i builder.cpp dodaj na górze linię komentarza: "Ver 1.1"
- Zaktualizowane pliki nanieś do właściwej gałęzi GitHub
- Odpowiedz na Pull requests
- Dodaj Project i kolumnę Done, a w niej napisz krótki raport wprowadzonych zmian  

3. Do gałęzi Projekt_STD nanieś odpakowaną zawartość dp1.zip
- Stwórz Pull request domagające się aktualizacji okna informacji
- Zaktualizuj powyższy projekt tak, aby w oknie informacje pojawiło się wasze imiona i nazwiska
- Zaktualizowany projekt nanieś do właściwej gałęzi GitHub
- Stwórz Pull request domagające się dodania informacji o wersji
- Pobierz projekt i w pliku dp1.cpp nanieś na górze linię komentarza: "Ver 1.0" 
- Zaktualizowany projekt nanieś do właściwej gałęzi GitHub
- Odpowiedz na Pull requests
- Dodaj Project i kolumnę Done, a w niej napisz krótki raport wprowadzonych zmian
